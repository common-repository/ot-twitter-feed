== Description ==
Twitter Feed is a simple WordPress plugin that displays a configurable number of updates (Tweets) based on the Twitter Widget Feed. Twitter Feed is twitter plugin which display your latest tweets. You just have to add your twitter widget ID and then you can see any userâ€™s tweets on your WordPress site. It is an awesome WordPress plugin to display your twitter tweets.

Twitter Update the easiest FREE solution for displaying Twitter Widget Feed for latest Wordpress version, it has 2 layouts: Slider layout and fixed layout

There is shortcode [ottwitter_feed] with 2 available parameters.

Visit plugin site

== Installation ==
1. Upload `ot-twitter-feed` zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Place the short-code [ottwitter_feed] to your theme files.
